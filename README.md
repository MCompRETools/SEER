This repository contains-
1. Training dataset for fine-tuning the sentence transformer model and the code for fine tuning. You can execute the code in any jupyter notebook platform. keep the dataset in the same folder as file.
2. Three set of catalogs that are required for execution of Relationship Integrator component.
3. The dataset and results obtained in the experimental process.
4. Source code for the three component of SEER framework and the source code for tool interface.
    a. You can execute the sustainability_interface.py in any jupyter notebook environment.
    b. In the environment, create 3 directories with uploads directory should contain the three python code files (SEER Tool source codes folder).
    c. Also you need to manually add the taxonomy csv into uploads folder. Download it from the repository mentioned in the article.
